This project performs sentiment analysis on Twitter data to classify tweets as positive or negative using Natural Language Processing (NLP) techniques and machine learning models.

##  Techniques Used
- Text Cleaning & Preprocessing
- TF-IDF Vectorization
- Logistic Regression (Baseline)
- Linear SVM (Optimized Model)
- Model Evaluation

##  Dataset
Sentiment140 Twitter Dataset containing labeled tweets for sentiment classification.

##  Model Details
- Baseline Model: Logistic Regression
- Optimized Model: Linear Support Vector Machine (SVM)
- Feature Engineering: TF-IDF with n-grams

##  Results
- Baseline Accuracy: ~76%
- Optimized Accuracy: ~77%

##  Conclusion
The optimized model demonstrated improved performance, highlighting the importance of feature engineering and model selection in NLP tasks.
