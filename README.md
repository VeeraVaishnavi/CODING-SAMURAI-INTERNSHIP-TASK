# Coding Samurai Internship Tasks

This repository contains the projects completed as part of the **Coding Samurai Internship Program**.  
The projects demonstrate my understanding of **Machine Learning, Deep Learning, and Natural Language Processing**, along with end-to-end project execution, evaluation, and documentation.

---

##  Projects Overview

###  Project 1: Customer Churn Prediction
**Description:**  
Built a machine learning model to predict whether a customer will churn based on behavioral and demographic features.

**Key Skills & Concepts:**
- Data preprocessing and EDA  
- Feature engineering  
- Classification algorithms  
- Logistic Regression and Artificial Neural Networks (ANN)  
- Model evaluation  

**Outcome:**  
Achieved strong predictive performance and demonstrated an end-to-end ML workflow.

---

###  Project 2: Image Classification Using Transfer Learning
**Description:**  
Developed an image classification system using **transfer learning** with a pre-trained **ResNet-50** model.

**Key Skills & Concepts:**
- Convolutional Neural Networks (CNN)  
- Transfer Learning  
- Data augmentation  
- Model training and evaluation  
- Performance analysis  

**Outcome:**  
Successfully implemented a deep learning pipeline and analyzed model behavior under computational constraints.

---

###  Project 3: Sentiment Analysis on Tweets
**Description:**  
Performed sentiment analysis on Twitter data to classify tweets into sentiment categories using **Natural Language Processing (NLP)** techniques.

**Key Skills & Concepts:**
- Text preprocessing and cleaning  
- TF-IDF vectorization  
- Logistic Regression (baseline)  
- Linear SVM (optimized model)  
- Model evaluation and comparison  

**Outcome:**  
Achieved ~77% accuracy after optimization, demonstrating effective feature engineering and model selection.
